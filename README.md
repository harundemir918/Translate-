# Translate!

**Translate!**, **React Native** ile yapılmış bir çeviri programıdır. Arkaplanda **Yandex Translate API** kullanmaktadır.

## Özellikler

<ul>
	<li>Kayıt-giriş işlemleri</li>
	<li>Kaynak-hedef dil seçimi</li>
	<li>Sesten metine çeviri (Speech-to-text)</li>
	<li>Metinden sese çeviri (Text-to-speech)</li>
</ul>

## Desteklenen Diller

<ul>
	<li>Türkçe</li>
	<li>İngilizce</li>
	<li>Almanca</li>
	<li>Fransızca</li>
	<li>İtalyanca</li>
	<li>İspanyolca</li>
	<li>Lehçe</li>
	<li>Rusça</li>
</ul>
